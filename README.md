# Interactive Quiz Platform

Interactive quiz platform built with Java, MySQL, HTML, CSS, and JavaScript.

## Tech stack
- Java, Spring Boot
- MySQL
- HTML, CSS, JavaScript

## How to run
1. Import `sql/quiz_platform_schema.sql` into MySQL.
2. Update `src/main/resources/application.properties` with your DB credentials.
3. Run backend: `mvn spring-boot:run`.
4. Open `Frontend/index.html` in a browser.
